# Full-stack showcase (GraphQl + React)

To build and launch the app along with setting up the API and DB clone the repository and type in the root folder:
```
build-up.bat
```

The application consists of:
- GraphQl API
- Mongo DB
- React frontend

Requirements:
- Docker
